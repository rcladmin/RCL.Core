# RCL Cloud Platform
The RCL Cloud Platform is comprised of core componenents to build line of buisness solution with the Microsoft Azure Cloud. 

## Contribute to this project

This section covers the process for contributing to the project.

- You need a basic understanding of Git and GitHub.com.

- Open an [issue](https://github.com/rcladmin/RCL.Core/issues) describing what you want to do. Wait for approval before you invest much time.

- Fork the repo of the **development** branch and create a new branch for your changes.

- Submit a PR to the **development** branch with your changes.

- Respond to PR feedback.
